# Glitch Drift

A visual text effect plugin for [Obsidian](https://obsidian.md) that applies glitch and distortion effects to selected text.

## Features

- **Real-time glitch effects**: Watch text glitch as you type
- **Adjustable intensity**: Control glitch intensity with a slider in settings
- **Lightweight**: Uses CSS animations, no heavy dependencies
- **Desktop-optimized**: Designed for desktop use

## Installation

1. Download the plugin from the Obsidian Community Plugins directory (coming soon)
2. Or: Clone this repo, run `pnpm install && pnpm build`, and copy `main.js`, `manifest.json`, and `styles.css` to your `.obsidian/plugins/glitch-drift/` folder

## Usage

Select text and use the command palette to apply glitch effects. Adjust the intensity in plugin settings.

## Development

pnpm install
pnpm dev        # Watch mode

pnpm lint       # Run linter
